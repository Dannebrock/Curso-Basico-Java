Bem-vindo ao repositório dos meus estudos em Java! 🚀

Este espaço foi criado para documentar e organizar todo o meu progresso prático durante o curso **"Java para Iniciantes - Fundamentos, POO e Projeto"** (ministrado na Udemy). 

⚠️ **Nota:** Este repositório não é um sistema complexo, mas sim uma **coleção de exemplos de código, snippets e exercícios isolados** que construí para consolidar a sintaxe básica e os pilares da Orientação a Objetos.

---

## 📚 O que você vai encontrar aqui

Os códigos estão organizados de acordo com as etapas do aprendizado:

* **Sintaxe Básica:** Variáveis, tipos de dados primitivos e operadores.
* **Primeiro Programa:** O clássico `Olá, Mundo!` e a estrutura inicial de um arquivo Java.
* **Estrutura de Controle:** Lógica de decisão (`if`/`else`) e repetição (`for`, `while`).
* **POO (Programação Orientada a Objetos):**
  * Criação de Classes e Instanciação de Objetos.
  * Atributos, Encapsulamento (`private`) e Métodos.
  * Construtores (padrão e com parâmetros).
* **Projeto:** Uma calculadora básica criada para colocar toda a teoria em prática.
